# 🎮 FUMIK — MASTER BUILD PROMPT
> Full-stack Multiplayer Gaming Platform

FUMIK is a real-time browser-based multiplayer gaming platform built with Next.js, Socket.io, and Matter.js.

## 🏗️ Tech Stack
- **Frontend:** Next.js 14 (App Router), Tailwind CSS, Framer Motion
- **Backend:** Node.js, Express, Socket.io
- **Physics:** Matter.js
- **Monorepo:** Turborepo, pnpm

## 🚀 Getting Started

1. **Install Dependencies:**
   ```bash
   pnpm install
   ```

2. **Run Development Mode:**
   ```bash
   pnpm dev
   ```
   This will start both the Next.js frontend (on port 3000) and the Socket.io server (on port 8080).

3. **Open in Browser:**
   - Go to `http://localhost:3000` to start playing.
   - Open in multiple tabs or devices to test multiplayer.

## 🎮 Included Games
1. **⚡ Reflex Royale:** Fast-fire reaction test.
2. **🧠 Brain War:** Speed-based trivia with cultural categories.
3. **🔵 Ball Blitz:** Physics-driven racing with Matter.js.
4. **🎨 Scribble Smash:** Creative emoji-based guessing game.
5. **🃏 Vote or Die:** Majority-rules social survival game.

## 🎨 Design System
- **Theme:** Chaotic & Loud (Neon Accents, Bold Typography)
- **Colors:** Hot Pink, Electric Yellow, Electric Blue, Loud Purple
- **Fonts:** Boogaloo, Space Grotesk, Space Mono

## 🔒 Security & Performance
- Rate-limited socket events
- Server-authoritative physics (Ball Blitz)
- Room auto-deletion after 30 mins of inactivity
- Mobile-first responsive design
